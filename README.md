# 🎓 MindPlay – AI Powered Student Learning Platform

MindPlay is a web-based AI-powered learning platform designed to help students revise topics, generate quizzes, and track their performance efficiently. It integrates AI features like summaries and mind maps to make learning smarter and interactive.

---

## 🚀 Features

### 👨‍🎓 Student Module

* User Registration & Login
* Interactive Quiz System (with timer)
* AI Topic Summary Generation
* AI Learning Assistant (Custom Topics)
* Mind Map Visualization
* Leaderboard & Performance Tracking

### 👨‍💼 Admin Module

* Manage Subjects & Syllabus
* Generate AI-based Quiz Questions
* View Student Data
* Dashboard Analytics

## 📦 Installation

### 1. Clone the Repository

```bash
git clone https://github.com/your-username/mindplay.git
cd mindplay
```

### 2. Setup Database

* Create database `mindplay_db`
* Import `database/schema.sql`

### 3. Configure Project

* Open `config/config.php`
* Add your database credentials
* Add your OpenRouter API key

### 4. Run Project

```bash
cd public
php -S localhost:8000
```

### 5. Access

👉 [http://localhost:8000](http://localhost:8000)
